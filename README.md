# ForensiQ Lite

Forensic audit intelligence platform for Indian CA firms.

## Vision
Reduce forensic investigation time using transaction intelligence and workflow automation.

## MVP Features
- Bank statement upload
- Transaction normalization
- Red flag detection
- Risk scoring
- Fund flow analysis
- Report generation

## Tech Stack
- Next.js
- NestJS
- PostgreSQL
- Prisma
