## Demo Flow (10 minutes)

1. **Dashboard** (1 min) - Show case overview
2. **New Case** (1 min) - Create investigation
3. **Import** (1 min) - Upload transactions
4. **Red Flags** (2 min) - Detection engine
5. **Fund Flow** (1.5 min) - Money trail analysis
6. **AI** (1 min) - Optional AI insights
7. **Entities** (1 min) - Party resolution
8. **Close** (30 sec) - Summary and Q&A
